# CONTRIBUTOR GUIDE

**Organization**: CRINE ([www.crine.in](https://www.crine.in))

---

## Technical Guidelines for Contributors

1. **Schema Integrity**: Changes to `schemas/` must remain backward compatible.
2. **Automated Testing**: Run `python validators/run_all.py` and `pytest tests/` before submitting PRs.
3. **Commit Messages**: Follow Conventional Commits format (`feat`, `fix`, `docs`, `refactor`).
