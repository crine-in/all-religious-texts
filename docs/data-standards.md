# DATA STANDARDS SPECIFICATION

**Organization**: CRINE ([www.crine.in](https://www.crine.in))

---

## Technical Data Standards

* **Unicode**: Standardized UTF-8 everywhere.
* **Transliteration**: Standardized IAST for Sanskrit, ISO 15919 for South Asian scripts, DIN 31635 for Arabic.
* **Checksums**: SHA-256 integrity hash required for all verse text fields.
