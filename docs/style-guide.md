# REPOSITORY STYLE GUIDE

**Organization**: CRINE ([www.crine.in](https://www.crine.in))

---

## Coding & Style Rules

* **Python**: Black formatting (line length 120), flake8 compliance, PEP 8.
* **JSON**: 2-space indentation, UTF-8, no BOM, deterministic key order.
* **Markdown**: GitHub Flavored Markdown, valid relative links, clear headings.
