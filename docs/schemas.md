# SCHEMA DOCUMENTATION

**Organization**: CRINE ([www.crine.in](https://www.crine.in))

---

## JSON Schemas (Draft 2020-12)

The ART platform provides 11 canonical JSON Schemas in `schemas/`:

* **`verse.schema.json`**: Individual verse representation.
* **`book.schema.json`**: Structural book metadata.
* **`translation.schema.json`**: Parallel translation alignment.
* **`commentary.schema.json`**: Scholarly exegesis and commentary.
* **`metadata.schema.json`**: Dataset folder metadata validation.
* **`license.schema.json`**: Copyright status verification.
* **`cross-reference.schema.json`**: Inter-textual links.
* **`contributor.schema.json`**: Contributor & scholar records.
* **`language.schema.json`**: ISO 639 language registry item.
* **`script.schema.json`**: ISO 15924 script registry item.
* **`edition.schema.json`**: Recension & manuscript tracking.
