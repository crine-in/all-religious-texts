# FREQUENTLY ASKED QUESTIONS (FAQ)

**Organization**: CRINE ([www.crine.in](https://www.crine.in))

---

### Q1: Why does `all-religious-texts` not contain full text datasets directly?
**A**: `all-religious-texts` is the core specification, schema authority, and validation hub for the ecosystem. Specific corpus datasets are maintained in dedicated repositories (`art-hinduism`, `art-buddhism`, `art-christianity`, etc.) to allow modular maintenance, individual versioning, and scalable repo sizes.

### Q2: How can I add a new translation or scripture?
**A**: Check [adding-scriptures.md](adding-scriptures.md) or open an issue on the relevant ecosystem repository.
