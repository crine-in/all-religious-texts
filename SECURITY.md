## Reporting a Vulnerability or Data Integrity Issue

Security concerns in our validation software, SDK packages, or dataset integrity verification scripts can be reported directly to the CRINE security team.

### How to Report:
* **Email**: `security@crine.in`
* **Response Time**: We acknowledge receipt of vulnerability reports within 48 hours and provide a fix timeline within 5 business days.

### Please Include:
* Type of issue (e.g. CLI path traversal, schema bypass, checksum spoofing, malformed regex exploitation).
* Detailed steps to reproduce.
* Affected scripts, tools, or schemas.

Do **not** open public GitHub issues for security vulnerabilities until a patch has been released.
